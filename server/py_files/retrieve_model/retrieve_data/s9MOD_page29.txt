Objectifs de la formation


Les bases de données sont au cœur de tout système d'information aujourd'hui omniprésent dans notre vie quotidienne (travail, organisation, web, etc.). Ce cours a pour objectif d'étudier les principes de programmation de bases de données relationnelles et semi-structurées qui sont les fondements de toute application dans les divers systèmes d'information. Il aborde aussi des aspects d'implémentation de systèmes de bases de données comme le contrôle de concurrence ou encore l'optimisation de requêtes.

Mots-clés : bases de données, modélisation de données structurées ou non, stockage et accès de données, langages relationnels, contrôle de concurrence, optimisation de requêtes, contraintes d'intégrité
